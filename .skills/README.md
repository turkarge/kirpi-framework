## Skills System

Kirpi Framework, geliştirme standardizasyonu için `.skills/` altında tanımlanan skill dosyalarını kullanır.

Bu dosyalar:
- framework kurallarını,
- UX davranış standartlarını,
- AI ile geliştirme disiplinini
tekilleştirir.

Yeni geliştirme yaparken ilgili skill dosyaları referans alınmalıdır.
