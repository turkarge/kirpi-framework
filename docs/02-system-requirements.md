# 02 - Sistem Gereksinimleri

## Minimum

- PHP 8.4+
- Composer 2+
- Docker + Docker Compose
- MySQL 8+
- Redis (opsiyonel ama onerilir)

## PHP Eklentileri

- pdo
- pdo_mysql
- mbstring
- json
- openssl
- zip (backup zip islemleri icin)

## Onerilen Gelistirme Ortami

- Docker tabanli local stack
- Windows + WSL2 / Linux / macOS
- Git

## Varsayilan Portlar

- `80` app
- `8081` manager
- `3306` mysql
- `6379` redis
