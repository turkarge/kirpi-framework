# 02 - System Requirements

## Minimum

- PHP 8.4+
- Composer 2+
- Docker + Docker Compose
- MySQL 8+
- Redis (opsiyonel ama onerilir)

## PHP Extensions

- pdo
- pdo_mysql
- mbstring
- json
- openssl
- zip (backup zip islemleri icin)

## Recommended Development Environment

- Docker-based local stack
- Windows + WSL2 / Linux / macOS
- Git

## Ports (default)

- `80` app
- `8081` manager
- `3306` mysql
- `6379` redis
